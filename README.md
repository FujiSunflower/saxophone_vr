# Saxophone VR
VRサックスです。他のVRソフトウェアで遊んでいる状態で使えます。  
Saxophone which you can use on playing other VR software. This is [Download](https://github.com/rn9dfj3/saxophone_vr/releases) and [Movie](https://youtu.be/8D1J7QPhH4o) !!!
